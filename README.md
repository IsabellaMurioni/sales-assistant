# Sales Assistant

## 📌 Descripción

Sales Assistant es una aplicación pensada para asistir en procesos de venta telefónica, brindando apoyo en tiempo real durante llamadas con clientes. La herramienta busca mejorar la comunicación, organización y efectividad de cada interacción comercial.

---

## 🎯 Objetivos

* Asistir al vendedor durante llamadas en tiempo real
* Guiar la conversación para aumentar las probabilidades de cierre
* Centralizar información clave del cliente durante la llamada
* Reducir errores y mejorar la claridad del discurso comercial
* Optimizar el rendimiento en ventas telefónicas
